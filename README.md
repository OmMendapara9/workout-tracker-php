# workout-tracker-php
# 💪 Workout Tracker App (Core PHP)

## 🚀 Overview

Workout Tracker is a modern fitness web application built using Core PHP and MySQL. It helps users track their daily workouts, monitor progress, and stay consistent with their fitness goals.

---

## 🎯 Problem

Many people go to the gym but fail to track their workouts properly.
This leads to:

* No progress tracking
* Lack of consistency
* Low motivation

---

## 💡 Solution

This app provides a simple and powerful dashboard where users can:

* Add workouts
* Track calories and duration
* Monitor progress with charts
* Mark workouts as completed

---

## ✨ Features

* 🔐 User Authentication (Login / Register)
* ➕ Add Workout (Activity, Duration, Calories)
* 📊 Dashboard with Analytics
* 📈 Chart.js Integration (Progress Tracking)
* ✏️ Edit Workout
* 🗑️ Delete Workout
* ✅ Mark Workout as Completed
* 🎨 Modern UI (Glassmorphism + Responsive Design)

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS, Bootstrap
* Backend: Core PHP
* Database: MySQL
* Charts: Chart.js

## ⚡ How to Run

1. Clone the repository
2. Import the database in phpMyAdmin
3. Update `config/db.php` with your database credentials
4. Run project in XAMPP / Localhost
